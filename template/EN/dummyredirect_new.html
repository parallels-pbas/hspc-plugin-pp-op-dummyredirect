The Dummy payment plug-in can be used for testing purposes (no data is sent to outside Parallels Business Automation - Standard).

Dummy Redirect Payment plug-in emulates a redirect plug-in in work. It creates a test page and redirects you there.

To configure the plug-in, fill the form:

Plug-in configuration.

Field Name
Description
Preauthorization enabled Select the type of transactions to process: if you want to process only the transactions of the Preauth type (two-step transaction consisting of money preauthorization on a credit card and money capture), then check the Preauthorization enabled box, if you want to process the Sale transactions only (one-step without preauthorization), then leave this box clear.

Plug-in setting

Field Name
Description
Server URL Specify the URL of the test page where Dummy Redirect redirects your request.

Miscellaneous settings. Here you can enter plug-in description.

Field Name
Description
Quick description of this payment method Enter the plug-in short description to be shown to your customers.
